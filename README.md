# UI-UX-Experiment2

## Aim:
To Design a Basic Mobile and Web Layout for an Existing Case Study.

## Algorithm:
Step 1: Analyze the Case Study

Identify users:

Customers ordering food

Restaurants managing orders

Delivery partners

Define core goals:

Browse restaurants

Order food quickly

Track delivery

List key screens:

Home

Search

Restaurant page

Cart

Order tracking

Profile

Step 2: Set Design Structure

Choose layouts:

Mobile-first design

Responsive web layout

Define:

Brand colors (warm, food-friendly tones)

Clean typography

High-contrast buttons

Step 3: Apply Gestalt Principles

For each screen:

• Proximity → Group food items with prices & ratings
• Similarity → Same button styles for “Add to Cart”
• Hierarchy → Highlight offers & popular dishes
• Alignment → Structured restaurant cards
• Continuity → Smooth flow from browse → order → payment

Step 4: Integrate UI Elements & Design Patterns

Add:

Search bar with filters

Restaurant cards

Floating cart button

Bottom navigation (mobile)

Sticky header (web)

Common patterns:

✔ Infinite scrolling
✔ Category tabs (Pizza, Biryani, Desserts)
✔ Quick reorder buttons
✔ Rating badges

Step 5: Develop Interaction Behaviors

Tap restaurant → open menu

Add item → cart updates instantly

Swipe to remove items

Live order tracking animation

Push notifications for delivery updates

Step 6: Apply Usability Principles

✔ Minimal steps to order
✔ Clear pricing breakdown
✔ Fast loading UI
✔ Easy navigation
✔ Error-free checkout

Step 7: Create Brand Logo in Canva

Choose food-related icon (plate, fork, flame, delivery bag)

Use bold readable font

Keep simple flat design

Export in PNG/SVG

Step 8: Integrate Logo into Layouts

Place on:

Splash screen

Header bar

Web homepage

Invoice screen

Maintain consistent size & spacing.

Step 9: Test User Experience

Perform usability testing

Measure:

Order completion time

Navigation clarity

Improve friction points

Step 10: Final Deliverables

✔ Mobile UI layout
✔ Web UI layout
✔ Logo design
✔ Interaction flow diagram
✔ Usability improvements
